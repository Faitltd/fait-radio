🧪 Testing the New Embedding Feature

How to test the new "Test Embed" feature:

  1. Click the "Admin Panel" button below to open the station manager
  2. Click "Add New Station" or edit an existing station
  3. Enter a YouTube video ID, playlist ID, or full YouTube URL
  4. Click the "Test Embed" button to check if the video allows embedding
  5. Watch the test results appear below the form
  6. If the test passes (✅), you can safely save the station
  7. If the test fails (❌), try a different video that allows embedding

Test Examples:

  • Should work: dQw4w9WgXcQ (Rick Astley - Never Gonna Give You Up)
  • Should work: jfKfPfyJRdk (LoFi Hip Hop Radio)
  • May fail: Many music videos from major labels don't allow embedding
